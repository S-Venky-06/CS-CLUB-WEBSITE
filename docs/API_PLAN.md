# API Plan

This document will describe the REST API endpoints for the Express backend.

## Purpose

Define the API contract between the Next.js frontend and Express backend:

- **Authentication endpoints**: Google OAuth login/logout/callback
- **Members endpoints**: Read member data from Google Sheets
- **Events endpoints**: Read event data from Google Sheets
- **Registration endpoints**: Submit registration forms to Google Sheets
- **Error handling**: Standard error response format
- **Rate limiting**: API usage limits
